# tic_tac_toe
A simple Tic-Tac-Toe game

```
************
Tic-Tac-Toe!
************

Please enter the column (A, B, or C) and then the row (1, 2, or 3) of your move.

   A B C
1 | | | |
2 | | | |
3 | | | |

Player Move (X): A3

   A B C
1 | | | |
2 | | | |
3 |X| | |

Computer Move (O): A2

   A B C
1 | | | |
2 |O| | |
3 |X| | |

Player Move (X): B3

   A B C
1 | | | |
2 |O| | |
3 |X|X| |

Computer Move (O): A1

   A B C
1 |O| | |
2 |O| | |
3 |X|X| |

Player Move (X): B2

   A B C
1 |O| | |
2 |O|X| |
3 |X|X| |

Computer Move (O): B1

   A B C
1 |O|O| |
2 |O|X| |
3 |X|X| |

Player Move (X): C3

   A B C
1 |O|O| |
2 |O|X| |
3 |X|X|X|

You(Player) win!
```
